# productivity-time-tracker-
C++ console application for tracking study/work time with daily, weekly, and monthly reports

## Features
**Activity Tracking**: Track time spent on each activity
**Comprehensive Reports**: Daily, Weekly, and Monthly summaries
**Data Persistence**: All sessions saved automatically after creation
**Activity Statistics**: View total time spent on each activity

## Technologies used
Language: c++ (c++20 modules)
Data Storage: File-based Applications and Vector Lists
Architecture: Object-oriented uml design

## Main Functionality
1. Start Tracking: Select an activity and start the timer
2. Stop Tracking: Stop the current session (automatically saved)
3. View Reports: Generate daily, weekly, or monthly reports
4. View All Activities: See cumulative time for all tracked activities
